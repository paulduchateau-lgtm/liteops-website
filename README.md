# LiteOps — Site Public

**Tier :** T1 — Public
**Stack :** Next.js 15 + Tailwind CSS
**URL :** [liteops-website.vercel.app](https://liteops-website.vercel.app)

Vitrine publique LiteOps : presentation des agents, operateurs, systemes et programme partenaires.

## Lancer en local

```bash
npm install
npm run dev
# http://localhost:3000
```

## Structure

```
src/app/          App Router Next.js
src/components/   Composants React (Navigation, AgentCard, etc.)
src/styles/       CSS Modules + globals
photosdesert/     Photos desert proprietaires
photosstaff/      Photos equipe
```
